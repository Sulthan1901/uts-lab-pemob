## ğŸ§¾ Deskripsi Aplikasi
**Quiz App** adalah aplikasi kuis interaktif berbasis **Flutter** yang memungkinkan pengguna untuk:
- Menjawab 10 pertanyaan pilihan ganda,
- Mendapatkan skor akhir,
- Melihat pembahasan setiap soal,
